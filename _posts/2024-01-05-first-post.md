<p class="intro"><span class="dropcap">M</span>utation testing is designed to ensure the quality of a software testing tool, not the applications it analyzes.</p>

Mutation testing is typically used to conduct unit tests. The goal is to ensure a software test can detect code that isn't properly tested or hidden defects that other testing methods don't catch.
